# Laboratorios de Infraestructura Cloud y DevOps ☁️

Este repositorio contiene las implementaciones técnicas realizadas durante mi formación en Cloud Computing, enfocadas en la automatización y orquestación de servicios.

## 🛠️ Tecnologías Aplicadas
- **AWS:** Configuración de servicios core (EC2, S3, redes).
- **Storage & Database:** S3, DynamoDB, RDS.
- **Networking:** VPC, Subnets, Route 53, API Gateway.
- **Contenedores:** Virtualización y empaquetado con **Docker**.
- **Orquestación:** Gestión de clústeres con **Kubernetes (EKS)**.

---

## 📁 Contenido del Repositorio
- **/Laboratorios_AWS:** Documentación y scripts de servicios de Amazon Web Services.
- **Lab_Orquestación_de_Kubernetes_EKS.md:** Guía detallada sobre el despliegue de nodos y servicios en EKS.
- **Laboratorio_de_contenedores_Docker.pdf:** Manual de creación y gestión de imágenes y contenedores.
